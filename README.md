# DistrictPortal
Автор: Осипов Арсений, Арсентьев Алексей 25ИС11Д

## Описание
Учебный проект: районный портал для жителей. Позволяет создавать объявления, следить за новостями района и взаимодействовать с соседями

## План функций
- Добавить объявление (пост) с категорией и описанием
- Просмотреть ленту объявлений с фильтрацией по категориям
- Комментировать объявления
- Управление профилем (указать свои навыки помощи)
- Отметить проблему на карте района
- Поиск по объявлениям

## Стек
C# / .NET / Visual Studio

## Как запустить
1) Открыть solution в Visual Studio
2) Нажать F5 (Run)
3) В консоли появится меню
